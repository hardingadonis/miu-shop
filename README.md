# Miu Shop

![GitHub repo size](https://img.shields.io/github/repo-size/hardingadonis/miu-shop)

> Miu Shop is a website that sells M.O.I cosmetics that are written in Java.


## Requirements
- JDK 8
- MySQL 8.1.0
- Tomcat 8.5
- Maven 3+
